## Definition

Common Criteria (ISO/IEC 15408) is an international framework for the evaluation and certification of IT security products. It defines Evaluation Assurance Levels (EAL1–EAL7) that represent increasing depth of evaluation, and Security Targets (ST) that define the security properties a product claims to implement.

## How It Works

A Common Criteria evaluation involves an independent laboratory assessing a Target of Evaluation (TOE) against its Security Target. Higher EAL levels require more rigorous analysis of design documentation, source code, and testing. Products achieving EAL certification can be listed on national trusted-product lists (US NIAP APL, UK NCSC CPA, etc.).

## Why It Matters for Network Security

Government and defense procurement frequently mandates Common Criteria evaluation for cross-domain solutions and cryptographic products. An EAL2+ evaluation provides documented evidence of design and implementation rigour that supports procurement and compliance decisions.
