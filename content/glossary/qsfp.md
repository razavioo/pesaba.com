## Definition

QSFP (Quad Small Form-factor Pluggable) is a high-density optical transceiver module supporting 4 independent data lanes. QSFP+ supports 4 × 10 Gbps (40 Gbps aggregate); QSFP28 supports 4 × 25 Gbps (100 Gbps aggregate); QSFP-DD supports 8 × 50 Gbps (400 Gbps aggregate).

## How It Works

QSFP modules use the same hot-pluggable form factor as SFP but with four transmit and four receive channels, multiplexed over a single fibre pair using wavelength-division multiplexing (WDM) or separate fibres. QSFP is used in high-speed data-centre interconnects, spine switches, and 40/100 Gbps links.

## Why It Matters for Network Security

High-speed network security appliances handling 40 Gbps or 100 Gbps links use QSFP interfaces. For data diodes at these speeds, the one-way optical barrier principle is the same as for SFP — only the module count and channel density differ.

## Related Pesaba Products

High-throughput Pesaba data diode variants supporting >1 Gbps use QSFP+ or QSFP28 interfaces.
