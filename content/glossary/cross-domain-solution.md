## Definition

A cross-domain solution (CDS) is a security-enforced information system that provides the ability to access data from or send data to different security domains. Data diodes are a hardware implementation of a CDS for environments where one-way information flow is sufficient.

## How It Works

A CDS enforces a defined security policy for data moving between two security domains — typically a higher-classification and a lower-classification network. The policy may include content inspection, data type filtering, format conversion, and audit logging. A hardware data diode implements the simplest and strongest form of CDS: data flows in one direction only, enforced by physics.

## Why It Matters for Network Security

CDS products are required wherever classified or sensitive data must be shared across security domain boundaries without allowing the lower-domain environment to access or influence the higher-domain environment. Government, defense, and high-assurance financial environments all have CDS requirements.
