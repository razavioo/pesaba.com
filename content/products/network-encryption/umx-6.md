## UMX-6 | Product Overview

UMX-6 is a compact desktop network encryption appliance designed and manufactured by Partov Ertebat Saba. The device is engineered for point-to-point Ethernet link encryption in government, military, and enterprise environments where data confidentiality on physical network segments is required.

Built into a low-profile steel chassis with a near-black finish, UMX-6 presents a minimal profile in sensitive installations. The front panel provides a single encrypted Ethernet port (ETH.ENC), a USB interface, and a console management port, keeping the operational footprint small and the attack surface minimal.

## Architecture

- **1 × ETH.ENC port** — encrypted 1 Gbps Ethernet link interface
- **1 × USB port** — key loading, firmware update, or peripheral interface
- **1 × Console port** — out-of-band management and configuration (RJ45)
- **Steel chassis** — rugged near-black enclosure for discreet deployment

## Key Features

- Single encrypted Gigabit Ethernet port (ETH.ENC) for point-to-point link encryption
- USB interface for key material loading and firmware management
- Console port for out-of-band management and configuration
- Compact desktop form factor — 140 × 120 × 30 mm
- Near-black steel chassis for discreet installation in sensitive environments
- Designed for government, military, and classified network deployments
