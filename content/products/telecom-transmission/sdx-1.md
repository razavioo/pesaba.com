## SDX-1 | Product Overview

SDX-1 is a 1U rack-mount SDH access multiplexer designed and manufactured by Partov Ertebat Saba. The device provides 2 × STM SFP optical ports alongside 3 × Gigabit Ethernet RJ45 access ports, a USB interface, and a console management port — enabling integration of Ethernet services into SDH network infrastructure and vice versa.

SDX-1 is designed for telecom access and aggregation nodes where legacy SDH transport must coexist with IP/Ethernet services. Eight front-panel LED indicators provide at-a-glance status for port link state and system alarms.

## Architecture

- **2 × SFP STM ports** — SDH optical line interfaces
- **3 × RJ45 Ethernet ports** — 1 Gbps access or uplink connections
- **1 × USB port** — configuration or firmware update interface
- **Console port** — out-of-band management (RJ45)
- **8 × LED indicators** — per-port and system status

## Key Features

- 2 × STM SFP optical ports for SDH network connectivity
- 3 × 1 Gbps RJ45 Ethernet ports for IP service access
- USB port for configuration and firmware management
- Console port for out-of-band management
- 8 front-panel status LEDs for rapid fault identification
- 1U rack-mount with standard rack ears
- Suitable for SDH/Ethernet service aggregation in carrier access networks
