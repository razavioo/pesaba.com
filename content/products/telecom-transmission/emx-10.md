## EMX-10 | Product Overview

EMX-10 is a 1U rack-mount SDH/STM-1 analyser, probe, and monitoring device designed and manufactured by Partov Ertebat Saba. The system is built around a single SFP optical port operating at STM-1 (155 Mbps) and a high-density array of 20 LED indicators for real-time channel status, alarm, and performance monitoring at a glance — without requiring a connected terminal for routine observation.

EMX-10 is built on FPGA architecture with no general-purpose processor or operating system in the data path, eliminating software-based attack vectors and ensuring predictable, deterministic behaviour on live SDH links.

## Architecture

- **1 × SFP port** — STM-1 optical interface (155 Mbps)
- **20 × LED indicators** — high-density status panel for channel-level and alarm visibility
- **Console port** — connection to management and analysis software
- **FPGA data path** — no OS, no processor in measurement path

## Key Features

- Single STM-1 / 155 Mbps SFP optical interface
- 20-LED high-density front-panel indicator array for real-time monitoring
- Console port for management software connection
- FPGA-based design — no operating system, no software attack surface
- EMX-10 Series cyan panel border for visual system identification
- 1U rack-mount with standard rack ears
- Suitable for carrier and government SDH/SONET network monitoring and analysis
