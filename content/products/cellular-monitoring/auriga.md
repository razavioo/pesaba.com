## Auriga | Product Overview

Auriga is a network quality evaluation system that is part of the Venus Pioneer suite. It collects the data required for network performance analysis by communicating directly with the modem of an operator or service provider — without requiring manual intervention from a test driver.

The system measures the actual end-user quality of experience (QoE) for data exchange, voice calls, and web/internet applications. All measured data is stored in the Venus Pioneer system and presented as real-time or historical (periodic) visual reports and output formats, based on a configurable time window.

Auriga is designed for passive, continuous, and automated network monitoring — suitable for deployment in vehicles, fixed sites, or as a remote probe.

## Key Features

- Drive testing without human supervision — no driver interaction required
- Frequency coverage mapping across all supported radio technologies
- Continuous remote monitoring and control of QoE and QoS
- Automatic identification of no-coverage and weak-coverage areas
- Video streaming performance analysis
- Geographic information display — status maps per city, province, or region
- Offline processing and geographic data export
- Direct installation on any vehicle or fixed chassis
- Signal strength and quality display for different test-path scenarios

## Supported Technologies

2G, GPRS, 3G (UMTS), 4G LTE, TD-LTE

## Integration

All Auriga measurement data flows to Venus Pioneer for aggregated reporting, historical trend analysis, and dashboard visualisation across the full network footprint.
